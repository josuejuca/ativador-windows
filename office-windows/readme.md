# 🏢 Ativação do Windows + Office

## 📋 Visão Geral
Script automatizado para ativar  **Windows** e **Microsoft Office** através de um único executável.

## ⚡ Guia Rápido

1. **Abra o arquivo** `ativa_todos_windows_e_office.bat` como **Administrador**
   - Clique direito no arquivo
   - Selecione "Executar como administrador"

2. **No menu do script:**
   - Pressione `1` para ativar o **Windows**
   - Pressione `2` para ativar o **Office**

3. **Aguarde a conclusão**
   - Uma mensagem **verde** aparecerá confirmando a ativação

## 📚 Mais Informações
- **Documentação oficial:** https://massgrave.dev/
- **Script base:** Microsoft Activation Scripts (MAS)

## ⚠️ Avisos de Segurança
- Execute **sempre como Administrador**
- Antivírus podem reportar falsos positivos

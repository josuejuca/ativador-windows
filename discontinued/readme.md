# ⚠️ Arquivos Descontinuados

Este diretório contém scripts de ativação que foram **descontinuados** e podem **não funcionar corretamente** em versões atuais do Windows.

## ⛔ Aviso Importante

Os arquivos nesta pasta são **legados** e podem apresentar os seguintes problemas:

- ❌ Incompatibilidade com versões recentes do Windows
- ❌ Métodos de ativação obsoletos
- ❌ Possível não funcionamento em Windows 10/11 **atualizados**
- ❌ Falta de suporte e manutenção

## 💡 O que fazer?

Se você está procurando scripts de ativação **funcionantes**, use os arquivos na **pasta raiz** do projeto:

- [`ativador.cmd`](https://github.com/josuejuca/ativador-windows/blob/main/ativador.cmd) - Script principal
- `office-windows/` - Scripts para Office

## 📋 Estrutura

Os arquivos originais descontinuados incluem:

- `office-windows/` - Scripts antigos para Office
- `win-10/` - Scripts para Windows 10 (versão antiga)
- `win8/` - Scripts para Windows 8 (descontinuado)

---

**Recomendação:** Use os scripts atualizados disponíveis na raiz do projeto.
